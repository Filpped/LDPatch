API 端点

获取发行版列表

GET /api/distributions

获取软件包列表

GET /api/packages/{distribution}

搜索软件包

GET /api/packages/search?q={query}&distribution={distribution}

获取软件包详情

GET /api/package/{distribution}/{package_name}

比较软件包

POST /api/compare

请求体示例:

{
    "distributions": ["Fedora", "openEuler-24.03"]
}

比较补丁

POST /api/compare/patches

请求体示例:

{
    "distributions": ["Fedora", "openEuler-24.03"],
    "packages_json": "common_packages.json"
}

快速开始

要比较两个发行版的软件包，请使用以下命令:

from relibrary.analysis.package_compare import compare_distribution_packages

# 比较Fedora和openEuler的软件包
result = compare_distribution_packages(['Fedora', 'openEuler-24.03'])

要比较补丁，请使用以下命令:

from relibrary.analysis.patch_compare import compare_patches_between_distros

# 比较Fedora和openEuler的补丁
compare_patches_between_distros(
    ['Fedora', 'openEuler-24.03'], 
    'common_packages.json', 
    'patch_comparison_report.json'
)
